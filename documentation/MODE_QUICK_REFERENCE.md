# Mode Selection Quick Reference

## At Startup
```
Enter mode [1-4] (default: 3/MEETING):

[1] QUICK MODE    - Personal, fast, no council
[2] WAR MODE      - Victory-focused, 5 ministers (Risk/Power/Strategy/Tech/Timing)
[3] MEETING MODE  - Balanced, 3-5 relevant ministers
[4] DARBAR MODE   - Deep wisdom, all 18 ministers
```
